If license error occurs or the PLCNext Engineer does not start:
Open CodeMeter Control Center, select Process and then Start Code Meter Service.
The Basic License should appear soon.

You can also double click the Basic License in this folder:
C:\Program Files\PHOENIX CONTACT\PLCnext Engineer 2024.0\Licenses

If there is an exception on start, it might be that you need to delete the following folder:
C:\Users\Administrator\AppData\Roaming\PHOENIX CONTACT
